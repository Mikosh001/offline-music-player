# Offline Music Player

Пайдаланушы телефон/ноутбук/iPhone-нан аудио файл таңдап қосады,
олар құрылғының өз жадында (**IndexedDB**) сақталады — серверге ешнәрсе
жіберілмейді. Интернет мүлдем болмаса да ойнатыла береді.

## Қалай іске қосу керек

Service Worker `file://` арқылы ашқанда жұмыс істемейді — локальды сервер керек.

```bash
cd offline-music-player
python -m http.server 8000
```
Браузерде: `http://localhost:8000`

Нақты құрылғыда (телефон/iPhone) сынау үшін — **GitHub Pages**-ке қой (README-дің
соңында нұсқау бар), себебі iOS Safari HTTPS емес адреске Service Worker-ге рұқсат
бермейді.

## Платформа бойынша ерекшеліктер

| Платформа | Файл қосу | Офлайн ойнату | "Қолданба" ретінде орнату |
|---|---|---|---|
| **Android (Chrome)** | Файл менеджерден таңдау | ✅ жұмыс істейді | Мәзір → "Негізгі экранға қосу" |
| **iPhone (Safari)** | Файлдар қолданбасынан таңдау | ✅ жұмыс істейді | Бөлісу → "Негізгі экранға қосу" |
| **Ноутбук/компьютер** | Файл диалогынан таңдау | ✅ жұмыс істейді | Chrome/Edge адрес жолындағы орнату белгішесі |

**iOS ескертуі:** Safari сайтты ұзақ уақыт (шамамен 7 күн) ашпасаң, сақталған
деректерді (соның ішінде IndexedDB-ді) автоматты тазалауы мүмкін. Бұл — iPhone-ның
жалпы саясаты, тек осы қолданбаға қатысты емес. Жиі қолданылатын сайттар үшін бұл
сирек болады.

## Қалай тексеру керек (офлайн)

1. Бетті ашып, "+ Ән қосу" арқылы 2-3 аудио файл қос
2. Бетті бір рет жаңарт (Service Worker кэшке толығымен жиналсын)
3. Wi-Fi/мобильді интернетті өшір (немесе DevTools → Network → Offline)
4. Бетті қайта аш — плейлист бұрынғыдай тұр, әндер ойнайды

## Файлдар құрылымы

| Файл | Рөлі |
|---|---|
| `db.js` | Ән файлдарын IndexedDB-де сақтау/оқу/өшіру |
| `script.js` | Плеер логикасы, транспорт түймелер, визуализатор |
| `service-worker.js` | Тек HTML/CSS/JS файлдарын кэштейді (әндерді емес) |
| `manifest.json` | Android/iOS-та "қолданба" ретінде орнатуға мүмкіндік береді |

## GitHub Pages-ке қою (тегін, HTTPS)

```bash
git init
git add .
git commit -m "offline music player"
git remote add origin https://github.com/Mikosh001/offline-music-player.git
git push -u origin main
```
Репозиторийдің **Settings → Pages** бөлімінен `main` бранчты таңда.
Сайт `https://Mikosh001.github.io/offline-music-player` мекенжайында
іске қосылады.

## Кеңес

- Файлдарды өзгерткен сайын `service-worker.js` ішіндегі `CACHE_NAME`-ды
  (`v1` → `v2`) өсіріп отыр
- Плейлист өте үлкен болса (жүздеген ән), браузер сұрайтын сақтау орнын
  ұлғайту диалогы шығуы мүмкін — құрылғы бос орнына байланысты
